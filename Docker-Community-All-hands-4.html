Docker API

  • docker API: https://docs.docker.com/engine/api/v1.41/
  • ping: https://docs.docker.com/engine/api/v1.41/#operation/SystemPing
curl --unix-socket /var/run/docker.sock http:/api/_ping
  • ps: https://docs.docker.com/engine/api/v1.41/#operation/ContainerList
curl --unix-socket /var/run/docker.sock http:/api/containers/json \
  | jq '.[]|[.Id]' -c
  • distribution: https://docs.docker.com/engine/api/v1.41/#operation/DistributionInspect
  • events: https://docs.docker.com/engine/api/v1.41/#operation/SystemEvents

docker-gen

renders a golang template file injecting all containers into the . context

cat > first.tmpl <<"EOF"
{{ range $key, $container  := . }}
  {{ $key }} : {{ $container.ID }}
{{ end }}
EOF

docker-gen first.tmpl

Docker-plugins

https://github.com/progrium/docker-plugins

list plugins

docker run --rm progrium/plugins plugn list

switch on autoremove and timeout plugins

docker run --rm -it \
	-v /var/run/docker.sock:/var/run/docker.sock \
	-e "ENABLE=autoremove timeout" \
	progrium/plugins
